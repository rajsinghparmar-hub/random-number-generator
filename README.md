# Random Number Generator (Python)

This is a simple Python program that generates a new random number every time it runs.

##  Features
- Generates a random number between **1000 and 99999**
- Uses Python's built-in `random` module
- Keeps running until the user chooses to stop

##  Requirements
- Python 3.x

##  How to Run
1. Clone the repository or download the file
2. Open terminal / command prompt
3. Run the program using:

```bash
python random_number.py
